# 按位异或的
**题目**:  
计算1^2^3^....^n之后的结果.  
**思路**:  
这个题目直接用遍历的话,也是可以的,但是由于n的大小不确定,所以时间复杂度为O(N),在这个题目中其实还是有可优化的空间的.  
我们列一下前次运算:  
```
1   0001    ->0001  1
2   0010    ->0011  3
3   0011    ->0000  0
4   0100    ->0100  4
5   0101    ->0001  1
6   0110    ->0111  7
7   0111    ->0000  0
8   1000    ->1000  8
```
