# 捡宝石  
一个寻宝者在一个山洞里面找到一堆宝石,宝石的种类多种多样,每种宝石都有自己的价值,寻宝者随身携带了一个可以给宝石称重,估价的设备.但是寻宝者只有一个最多可以装重量为M的宝石.现在设每块宝石的重量分别为wi(i=0,1,2,3,4,5...),价值分别为vi(i=0,1,2,3,4,5,6....).求背包能够容纳宝石的最大价值.  
**输入**:
```
N M
w1 v1
...
wn vn  
```
`N`为石头的总数,`M`为背包最大可以容纳的重量.  
**输出**:  
```
最大的价值数
```  
**示例**:  
**输入**:
```
5 10
1 2
3 2
6 4
5 5
3 3
```
**输出**:
```
10
```
