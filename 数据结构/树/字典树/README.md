# 字典树  
顾名思义,采用字典索引的方式查询指定字段的值  
将字符串的每个字符作为一个节点  
e.g
apple   苹果
boy     男孩
app     应用
application 应用
                                字符位置
a  ->       b                       1
p           o                       2
p(应用)     y(男孩)                 3
l                                   4
e(苹果) ->  i                       5
            c                       6
            a                       7
            t                       8
            i                       9
            o                       10
            n(应用)                 11

相同位置的字符处于同一层(比如apple和boy中,a和b都是第一个字符,所以他们处于同一层)
