# grep的替代命令
以前需要查询包含某个字符串可以用grep命令
比如:`grep -ri 需要查找的字符串 目录`
现在在linux中有两个更好用的工具:`ag`和`ack`
需要查询当前目录下包含某个字符串的文件可以执行:  
`ag 字符串`或`ack 字符串`  
需要查询指定目录下包含指定字符串的文件只需要执行:  
`ag 字符串 目录`或`ack 字符串 目录`  
指定-i选项可以忽略字符串的大小写  
e.g.  
```
ag apple    # 查询当前文件夹下包含apple的文件
ag apple /root/ # 查询/root文件夹下包含apple的文件
ag -i apple /root/  # 查询/root文件夹下包含apple的文件,忽略大小写
```
经过简单的测试,ag的速度比ack的查询速度快很多.可能是因为ag的名字短一点的缘故(手动滑稽)
